Metamaterial Generator — Diffusion Model

This runs the trained conditional diffusion model (DDPM) directly in your browser via ONNX Runtime Web. You set a target volume fraction and stiffness class; the model denoises new unit cells, and each one is scored live with real physics: exact volume fraction, periodic connectivity, and a numerical homogenization for effective stiffness and Poisson's ratio.

Targets

First click loads the 8 MB model. Stiffness and Poisson are relative to the solid material and homogenized at 24×24 for speed (close approximations, not exact).
How it works & limits

The ONNX graph is the denoiser; the DDIM loop, classifier-free guidance, and the evaluator all run in JavaScript. The numpy-equivalent pipeline matches the PyTorch sampler at 0.9989 IoU.

Volume-fraction control is calibrated and reliable in roughly 0.37–0.57 (the model's response is monotonic but compressed). Stiffness conditioning separates low vs high by about 2×. There is no auxetic control: the training set has essentially no negative-Poisson cells, so ν is reported but not targetable.
