Metamaterial Unit-Cell Generator

Set a target, and the page generates candidate periodic unit cells, then evaluates each one with real physics computed live in your browser: exact volume fraction, periodic connectivity, and a numerical homogenization (conjugate-gradient FEM) for effective stiffness and Poisson's ratio. It keeps the designs that best match your target. This mirrors the project's generate-then-filter method.

Targets

Stiffness and Poisson are relative to the solid material (solid = 1.0). Homogenization runs at 24×24 with a softened void for speed, so values are close approximations of the full-resolution solver, not exact.
How it works & honest limits

Candidates come from a stochastic generator (smoothed Gaussian-random-field topologies plus strut/lattice motifs), optionally folded to enforce symmetry. Each is scored by the same homogenization algorithm used to label the project dataset, ported to JavaScript and validated against the Python solver.

Auxetic note: genuinely negative Poisson's ratio needs slender, well-proportioned re-entrant lattices that a coarse pixel grid can't represent, so the "Min Poisson" goal searches for the lowest value it can find and reports the real number. It is usually still positive. True auxetic designs need finer re-entrant geometry or real topology-optimization data, which is the documented limit of this project.
